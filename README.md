# Z4HL3N
Z4HL3N ist ein einfachs Spiel, bei dem ausgeschriebene Zahlen vorgegeben werden, die der Spieler ziffernweise eingeben muss.
Jeder korrekte Versuch wird gezählt.
Wieviele Zahlen schaffst du in 600 Sekunden?
Viel Spaß!
